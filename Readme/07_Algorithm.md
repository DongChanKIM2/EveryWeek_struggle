## 정렬
- Bubblesort(쉽고 편리 but 시간복잡도: n^2)
- Countingsort(최대값을 알고 작은수가 여러개 반복될 때 사용, 시간복잡도: n)
- 선택정렬

## 패턴매칭
- BruteForec(갯수 탐색도 가능)



## append vs extend

- append: original에 새로운 존재[]를 추가하는 것
- extend: original 내부에 순차적으로 추가해주는 것

- 분할정복 알고리즘(quick sort)
